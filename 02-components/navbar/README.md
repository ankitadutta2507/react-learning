# Navbar Component

## Overview

This is my first guided React component built while learning the fundamentals of React.

The component renders a simple navigation bar with a nested submenu using semantic HTML and JSX.

## Concepts Practiced

- Functional Components
- JSX
- Component Export
- Nested Elements
- Semantic HTML (`<nav>`, `<ul>`, `<li>`)
- Accessibility Attributes
  - `aria-expanded`
  - `aria-label`
- `className` in React

## Features

- Navigation menu
- Nested submenu for Apps
- Accessible markup
- Clean and reusable component structure
